# 🌸 Portfolio Website – Miniproject PWB

Website ini merupakan project portfolio pribadi yang dibuat untuk memenuhi tugas **Mini Project Pemrograman Web Berbasis (PWB)**.
Website ini menampilkan informasi profil, pengalaman, serta sertifikat yang dimiliki.

## ✨ Fitur Website

* Menampilkan **profil mahasiswa**
* Menampilkan **deskripsi diri**
* Menampilkan **sertifikat**
* Data diambil secara **dinamis dari database MySQL**

## 🛠 Teknologi yang Digunakan

* **HTML**
* **CSS**
* **Bootstrap**
* **PHP**
* **MySQL**
* **Laragon** (sebagai local server)

## 📂 Struktur Project

```
portfolio-website
│
├── index.php
├── koneksi.php
├── style.css
├── images/
│   ├── Aplikasi25.jpg
│   ├── dicoding.png
│   └── studyclub.png
```

## 🗄 Database

Database yang digunakan bernama:

```
portfolio_db
```

Tabel utama:

* **profile** → menyimpan nama dan deskripsi profil
* **certificates** → menyimpan data sertifikat

## 🚀 Cara Menjalankan Project

1. Jalankan **Laragon**
2. Import database ke **phpMyAdmin**
3. Letakkan folder project di:

```
laragon/www/
```

4. Buka di browser:

```
http://localhost/nama-folder-project
```
